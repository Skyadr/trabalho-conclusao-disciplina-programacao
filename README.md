Trabalho de conclusão da disciplina de programação com javascript, onde foi solicitado criar uma classe de serviços de pagamento, onde há dois métodos: 1 método para armazenar pagamento e 2 padrões distintos, "caro" para pagamentos > 100.00 e "padrão" para pagamentos <= 100.00; e 1 método para consultar o último pagamento efetuado. Os testes utilizando mocha e asserts do node validam os devidos padrões lançados.
